## Ai Nanika (AI何か)

デスクトップ上にキャラクターを召喚し、
OpenAI APIを通じてリアルタイムに会話を楽しむことができるElectronアプリケーションです。  
デスクトップに常駐し、作業の傍らでAIキャラクターとの生活を楽しめます。  
<br><br>
## 🌟 主な特徴
デスクトップ常駐型: キャラクターがデスクトップに居座り、あなたの作業を見守ります。  
OpenAI API連携: 最新のAIモデルを活用した高度な会話が可能です。  
高いカスタマイズ性: キャラクターの人格（persona.txt）や話題（topics.json）を自由に入れ替えることができます。  
プライバシー重視: APIキーはローカルの安全な領域に保存され、外部に送信されたりリポジトリに含まれたりすることはありません。  
<br><br>
## 🚀 使い方（一般ユーザー向け）  
アプリの起動配布されたアプリケーション（DMG等）をインストールし、起動します。  
APIキーの設定初回起動時、または保存済みのAPIキーが無効な場合、入力ダイアログが表示されます。自分の OpenAI APIキー を入力してください。  
会話の開始APIキーの認証が完了すると、キャラクターとの会話が始まります。  
<br><br>
## ⚠️APIキーの保存について⚠️  
入力したAPIキーはアプリ内のローカル保存領域（Application Support）に保存されます。  
GitHubのリポジトリや配布ファイルに含まれることはありませんのでご安心ください。  
<br><br>
## 🛠️ 開発者向けガイド  
環境構築依存関係をインストールします。
```bash
npm install
```
開発用起動ローカル環境でアプリを起動します。
```bash
npm start
```
パッケージング配布用のディスクイメージ（DMG）を作成します。
```bash
npm run dist
```

生成されたファイルは dist/ ディレクトリに出力されます。
<br><br>
## 📁 プロジェクト構成（主要ファイル）  
renderer.js：メインのロジック、AI呼び出し、UI処理  
main.js：Electronのメインプロセス、ウィンドウ管理  
persona.txt：キャラクターの性格・口調の設定  
topics.json：自動発話（独り言）の話題リスト  
messages.json：システムメッセージ（起動時・終了時など）の定義
